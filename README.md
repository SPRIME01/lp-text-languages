lp-text-langauge
================

  lp-text-langauge provides a declarative way with HTML attributes to show texts
  translated.

Usage
=====

```
<body unresolved>

        <div id="mylanguage"></div>

        <lp-text-languages
          pt-br="Um texto escrito nos seu idioma preferido!"
        >
          A default text wrote in your preference language!
        </lp-text-languages>

</body>
```
